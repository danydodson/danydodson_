## :checkered_flag: Starting

```bash
# Clone this project
$ git clone https://github.com/danydodson/danydodson.github.io

# Access
$ cd danydodson.github.io

# Install dependencies
$ yarn

# Run the project
$ yarn start

# The server will initialize in the <http://localhost:8000>
```
